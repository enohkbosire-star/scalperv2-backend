🔄 THREE OPERATING MODES
═══════════════════════════════════════════════════════════════════════════════

MODE 1: BACKTEST (Test Strategy on Historical Data)
────────────────────────────────────────────────────
Input:   CSV file with 500-2000 candles
Process: Simulates trades on past data
Output:  Performance metrics & statistics

Results Example:
  Total Trades: 47
  Win Rate: 65.96%
  Profit Factor: 2.84
  Net Profit: $1,247.50

Use Case: Validate strategy before live trading


MODE 2: LIVE TRADING (Automated MT5 Integration)
────────────────────────────────────────────────
Input:   Real-time MT5 data from chart
Process: 
  1. MT5 detects signals
  2. Sends to Java API
  3. Bot processes signal
  4. Positions automatically opened
  5. Telegram alert sent

Output:  Live P&L, executed trades, alerts

Use Case: Automated 24/5 trading


MODE 3: SIGNAL GENERATION (Manual Trading)
──────────────────────────────────────────
Input:   Historical or real-time data
Process: Generates trading signals
Output:  CSV file with signal details

Signal Output:
  Direction | Entry | SL | TP | Confidence | Strength | Reason

Use Case: Manual trading or UI integration
